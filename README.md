# 🔤 Jogo de Adivinhar Palavras

<img width="710" height="848" alt="image" src="https://github.com/user-attachments/assets/bbca42ea-bb6a-4a34-9203-8f17127d8cd9" />


Projeto desenvolvido com o objetivo de criar um **jogo de adivinhação de palavras** em uma aplicação web utilizando **React**.

A proposta do projeto foi aplicar os principais conceitos aprendidos em React até o momento, construindo uma experiência interativa com sistema de dicas, histórico de tentativas, validações e reinício de partida.

---

## 🧠 Sobre o projeto

O jogo permite ao usuário:

- 🎯 Tentar adivinhar uma palavra secreta
- 💡 Receber dicas durante a partida
- 📝 Visualizar o histórico de tentativas
- ⚠️ Receber validações para entradas inválidas
- 🔄 Reiniciar o jogo a qualquer momento
- 📱 Utilizar a aplicação em diferentes dispositivos

O layout da aplicação foi baseado em uma referência desenvolvida no **Figma**.

---

## 🧠 O que foi trabalhado

Durante o desenvolvimento, foram aplicados conceitos como:

- ⚛️ Criação de componentes com React
- 🧠 Gerenciamento de estado
- 🎯 Manipulação de eventos
- 🔁 Renderização dinâmica de informações
- 📝 Histórico de palpites do usuário
- ⚠️ Validações de entrada
- 🔄 Reinício da aplicação e controle de fluxo do jogo
- 📱 Estruturação de interface responsiva

---

## 🚀 Tecnologias utilizadas

- React
- TypeScript  
- HTML5  
- CSS3  

---

## 🎯 Objetivo do projeto

Este projeto tem caráter **educacional**, com foco em:

- Praticar os fundamentos do React
- Trabalhar gerenciamento de estado e reatividade
- Construir aplicações interativas no front-end
- Desenvolver lógica de jogo e experiência do usuário
